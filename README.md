# 🗳️ Sistema de Votaciones

Este proyecto es un ** sistema de votaciones** desarrollado como parte del proceso de formulación del proyecto académico.

---

## 🧠 Descripción

El sistema de votaciones permite a los usuarios:

- Iniciar sesión en el sistema  
- Emitir su voto de manera segura  
- Consultar el registro de votos  
- Acceder a un panel de administración  
- Eliminar o modificar votos registrados  

El diseño se enfoca en ofrecer una **experiencia intuitiva, accesible y moderna**, validada mediante pruebas de usabilidad.

---

## 💻 Tecnologías Utilizadas

- **Angular** – Framework para el desarrollo del frontend  
- **Firebase** – Autenticación y base de datos en la nube  
- **Figma** – Herramienta de diseño de interfaces  
- **Maze** – Plataforma para pruebas de usabilidad
