# Sistema de Votaciones 🗳️

Este proyecto es un prototipo funcional de un sistema de votaciones desarrollado como parte del proceso de formulación del proyecto.

## 🧠 Descripción
El sistema permite a los usuarios:
- Iniciar sesión
- Emitir votos
- Consultar resultados
- Administrar votos desde un panel de control

## 👨‍💻 Tecnologías utilizadas
- Angular
- Firebase
- Figma (para el diseño)
- Maze (para pruebas de usabilidad)

## 📄 Documentación
El documento **"Documento de formulación del proyecto.docx"** contiene los detalles del análisis y las pruebas de usabilidad.
